How Did We Transform The Data?

Step 1

In order to quickly grab the data corresponding to the ‘Sample Name’ and ‘Irradiated/not-irradiated’ columns, a web scraping algorithm was created to be used for all 3 datasets. In this program, a python WebDriver library called Selenium was used to access/open the website of choice and pass the page source/HTML to another library called BeautifulSoup4. This second python library would read and save all the relevant data from the aforementioned columns. In the cases where the table contained more rows than what was currently displayed, Selenium was used to click the ‘next’ button to load the remaining rows.

Step 2

There were a lot of nested hashmaps and lists within the JSON file so extracting the needed data (e.g. %GC and %AT) was difficult.

Step 3

Once the data was both scraped and parsed, it was saved in the form of a python dictionary or hashmap where every key represented a specific JSON file attribute (e.g. %GC before) and every value was a list containing all the values for that key for every sample.

Final Step

We created a normal distribution for every ‘before being in space’ and ‘after being in space’ column

We then Compared the normal distributions to find %change in GC before and after space using Cohen’s D

We used Cohen’s D. of two normal distributions: P1(μ1,σ1) and P2(μ2,σ2) is (μ2-μ1)/(σ1*σ2)

For every key-value pair in the dictionary as outlined above, a normal distribution was created to display the probabilities of the specific attribute (e.g. %GC before) being a certain value. Now to address our space biology question as to whether/how space flight would affect ATCG levels in animals/plants, related distributions (e.g. %GC before and %GC after) were compared using Cohen’s D values. Cohen’s D of two normal distributions P1(μ1,σ1) and P2(μ2,σ2) is calculated as (μ2-μ1)/(σ1*σ2).

Final Results

FINAL RESULTS: The final results above show that there was a notable difference in %change for AT, GC, and duplicates in the DNA but not a significant change in the number of total_sequences across all datasets. Cohen’s D values close to 0 indicate not much change between the before space and after space values. A positive Cohen's d value indicates that the mean of the first group (usually the treatment group) is greater than the mean of the second group (usually the control group). The larger the positive Cohen's d value, the larger the effect size, and the more substantial the difference between the two groups. A negative Cohen's d value indicates that the mean of the first group is smaller than the mean of the second group.
